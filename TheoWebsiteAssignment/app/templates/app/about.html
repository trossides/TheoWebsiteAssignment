Describing my Data Model

My data model has a table named 'Club'. I created a class in my model.py file which declaired this table and the fields it contains. I am using the SQL lite database which is defined in my settings.py file. I have in total 6 fields within my 'Club' class. The Primary Key (PK) for my 'Club' class was auto incremented thanks to the Django framework. The PK is an integer and is set to NOT NULL.

Below you can find the name for each of the 6 fields within my 'Club' class with a description:

name

What is it called: name

What is its purpose: Name of the Club

What data type: CharField (varchar)

Validation or constraint requirements: NOT NULL, Max Length of 50 characters, Has a verbose_name property

year_formed

What is it called: year_formed

What is its purpose: The year the Club was formed

What data type: PositiveIntegerField (integer)

Validation or constraint requirements: NOT NULL

president_name

What is it called: president_name

What is its purpose: The president name of the Club

What data type: CharField (varchar)

Validation or constraint requirements: NOT NULL, Max length of 25 characters

email

What is it called: email

What is its purpose: The contact email address for the Club

What data type: EmailField (varchar)

Validation or constraint requirements: NOT NULL, Has an auto character limit of 254

phone_number

What is it called: phone_number

What is its purpose: The phone number to contact the Club

What data type: CharField (varchar)

Validation or constraint requirements: NOT NULL, Max length of 12 characters

club_nickname

What is it called: club_nickname

What is its purpose: The Nickname of the Club

What data type: CharField (varchar)

Validation or constraint requirements: NOT NULL, Max length of 20 characters
